Intégration Jenkins + Ansible

TP6 - this is som Déploiement automatisé d’un projet depuis GitHub

Présentation du TP

Ce projet montre comment utiliser Jenkins pour automatiser l'extraction d’un projet depuis GitHub, et Ansible pour déployer ce projet sur un serveur cible. L’ensemble constitue une chaîne CI/CD simple et efficace.

Workflow

  1. ✅ Jenkins est déclenché manuellement ou automatiquement
  2. 📦 Récupération du code source depuis GitHub
  3. 🔧 Ansible est appelé par Jenkins pour exécuter le déploiement
  4. 🚀 Le projet est déployé sur une machine distante

Outils Utilisés

  • 🔧 Jenkins — Automatisation des tâches CI/CD
  • 🧰 Ansible — Orchestration et configuration distante
  • 💻 GitHub — Hébergement du code source
  • 🐧 Linux — Environnement de déploiement

Conclusion

Cette intégration montre la puissance des outils DevOps modernes pour automatiser le cycle de vie du développement logiciel. Grâce à Jenkins et Ansible, le processus de livraison devient fluide, répétable et sécurisé.
